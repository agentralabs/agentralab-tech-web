## LinkedIn Banner

- Template SVG: `agentra-linkedin-banner-template.svg`
- Upload-ready PNG: `agentra-linkedin-banner-template.png`
- Upload-ready JPG: `agentra-linkedin-banner-template.jpg`
- Logo slot image: `agentra-linkedin-logo-slot.jpg`

### Replace logo

1. Replace `agentra-linkedin-logo-slot.jpg` with your own JPG logo (keep same filename).
2. Re-export the banner from SVG to PNG/JPG if needed.

Recommended LinkedIn banner size used here: `1584 x 396`.
