## Key capabilities

  <FeatureCard icon="brain" title="AgenticMemory" description="Persistent cognitive graph memory with portable .amem artifacts, causality edges, and cross-session continuity." />
  <FeatureCard icon="eye" title="AgenticVision" description="Browserless web perception and visual continuity with .avis artifacts, mapping, comparison, and action surfaces." />

## AgenticMemory

<FeatureGrid>
  <FeatureCard icon="search" title="Graph navigation" description="Memory retrieval by traversal, not flat chunk similarity." />
  <FeatureCard icon="git" title="Truth revision" description="Corrections and supersession edges preserve reasoning history with explicit state evolution." />
  <FeatureCard icon="gauge" title="Operational speed" description="Memory-mappable binary format optimized for low-latency access." />
  <FeatureCard icon="book" title="Portable artifact" description="Single-file .amem memory that moves across providers and sessions." />
</FeatureGrid>

<FeatureGrid>
