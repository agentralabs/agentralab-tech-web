<Callout type="info" title="Integration model">
Each project is standalone-first. You can run Memory-only, Vision-only, or Codebase-only. Combined operation is optional through MCP.
</Callout>

## Auto-merge clients

Installer scripts can merge supported local config for:

- Claude Desktop
- Claude Code

## Find MCP config on desktop after install

If you installed successfully but cannot find where MCP was written, use these checks:

<CommandTabs
  title="Locate MCP config files"
  description="Run the channel that matches your platform."
  items={[
    {
      label: "macOS/Linux",
      code: "find \"$HOME\" -maxdepth 5 \\( -name \"*mcp*.json\" -o -name \"claude_desktop_config.json\" \\) 2>/dev/null | head -n 60",
      description: "Lists common MCP/desktop config files under your home directory.",
    },
    {
      label: "Search entries",
      code: "rg -n \"agentic-memory|agentic-vision|acb-mcp|agentra\" \"$HOME/.claude\" \"$HOME/.config\" \"$HOME/Library/Application Support\" 2>/dev/null",
      description: "Finds files containing Agentra MCP entries.",
    },
  ]}
/>

<Callout type="tip" title="After install behavior">
If the installer reports merge success, restart the desktop client fully, then run one MCP tool call to confirm registration.
</Callout>

## Manual MCP entry (other clients)

For Cursor, VS Code, Windsurf, Cline, Codex Desktop, and generic MCP clients:

<Command
  title="MCP server entry"
  description="Add one server entry per runtime in your client config file."
  code={`{
  "command": "/Users/<you>/.local/bin/agentic-vision-mcp",
  "args": ["serve"]
}`}
/>

Equivalent binaries:

- `agentic-memory-mcp`
- `agentic-vision-mcp`
- `acb-mcp`

## Validation checklist

<Command
  title="Binary path validation"
  description="Run this before debugging client-side connection issues."

which acb

## Safeguard MCP config

<Command
  title="Backup MCP config before manual edits"
  description="Create a timestamped backup so you can roll back quickly."
  code={`mkdir -p \"$HOME/.agentra-backups/mcp\"\nif [ -f \"$HOME/.claude/mcp.json\" ]; then\n  cp \"$HOME/.claude/mcp.json\" \"$HOME/.agentra-backups/mcp/claude-mcp-$(date +%F-%H%M%S).json\"\nfi`}
/>
