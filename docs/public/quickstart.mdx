<Callout type="tip" title="Zero-friction setup">
Use one installer command per runtime. Select the profile that matches your environment:
- `desktop` for Claude Desktop/Claude Code auto-merge
- `terminal` for binary-only local CLI installs
- `server` for remote hosts and service environments
</Callout>

## Prereqs

- macOS, Linux, or Windows with WSL
- `curl` available
- Rust toolchain installed (fallback source build path)

<Steps>
  <Step number={1} title="Install one runtime (or all three)">
    <CommandTabs
      title="Desktop profile (default/backward-compatible)"
      description="Desktop profile auto-merges supported local MCP desktop configs."
      items={[
        {
          label: "AgenticMemory",
