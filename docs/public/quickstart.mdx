## Before you begin

- macOS, Linux, or Windows with WSL
- `curl` available
- Rust toolchain installed for fallback source builds

## 1. Install AgenticMemory

```bash
curl -fsSL https://agentralabs.tech/install/memory | bash
```

## 2. Install AgenticVision

```bash
curl -fsSL https://agentralabs.tech/install/vision | bash
```

## 3. Install AgenticCodebase

```bash
curl -fsSL https://agentralabs.tech/install/codebase | bash
```

## 4. Verify binaries

```bash
amem --version
agentic-vision-mcp --version
acb --version
acb-mcp --version
```

## 5. Verify MCP runtime (optional)

```bash
agentic-memory-mcp --help
agentic-vision-mcp serve --help
acb-mcp serve --help
```
