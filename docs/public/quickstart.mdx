<Callout type="tip" title="Zero-friction path">
Use one installer command per project. Each installer handles platform detection, binary install, and MCP merge for supported clients.
</Callout>

## Before you begin

- macOS, Linux, or Windows with WSL
- `curl` available
- Rust toolchain installed (fallback source install)

## 1. Install AgenticMemory

<Command
  title="Install AgenticMemory"
  description="Adds amem runtime + memory MCP server and updates compatible local client config."
  code={`curl -fsSL https://agentralabs.tech/install/memory | bash`}
/>

<CliOutput
  title="Expected signal"
  output={`[ok] Platform: darwin-aarch64
[info] Configuring MCP clients...
[ok] Done! Memory defaults to ~/.brain.amem`}
/>

## 2. Install AgenticVision

<Command
  title="Install AgenticVision"
  description="Installs vision MCP runtime for browserless visual workflows and merges local MCP config."
  code={`curl -fsSL https://agentralabs.tech/install/vision | bash`}
/>

## 3. Install AgenticCodebase

<Command
  title="Install AgenticCodebase"
  description="Installs acb + acb-mcp for semantic code intelligence workflows."
  code={`curl -fsSL https://agentralabs.tech/install/codebase | bash`}
/>

## 4. Verify installed binaries

<Command
  title="Binary checks"
  description="Confirm all major binaries are available in your shell PATH."
  code={`amem --version
agentic-vision-mcp --version
acb --version
acb-mcp --version`}
/>

## 5. Optional MCP runtime checks

<Command
  title="Runtime sanity check"
  description="Useful when validating client integration or troubleshooting connection issues."
  code={`agentic-memory-mcp --help
agentic-vision-mcp serve --help
acb-mcp serve --help`}
/>
