## 工作区编排器契约

- 姐妹项目必须保持独立可安装、可运行。
- `agentra`（或未来编排器）仅是可选 UX，不得成为硬依赖。
- 编排器 UX 必须探测已安装工具，并给出可执行的缺失状态提示。
- 编排器 UX 可以提供状态与提示，但不得静默强制捆绑安装。
- 若存在共享 workspace，相关命令必须保持可用。

## 编辑守则

- 保持官方命名：`AgenticMemory`、`AgenticVision`、`AgenticCodebase`
- 声明必须与已发布能力一致
- 基准表述必须对应可复现来源
- 路线图敏感内容不放公开文档
