## 运行时截图

下面把运行过程拆成小块（chunk），每块都按同一顺序：
1. 复制命令。
2. 运行命令。
3. 对照输出。
4. 查看截图。

<Callout type="tip" title="阅读方式">
按顺序阅读每个 chunk。每块都有命令、输出、截图和简单解释。
</Callout>
