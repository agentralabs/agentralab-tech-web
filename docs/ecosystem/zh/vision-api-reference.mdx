## MCP 工具

- `vision_capture`：捕获并嵌入图像。
- `vision_query`：通过过滤器查询捕获。
- `vision_similar`：查找视觉上相似的捕获。
- `vision_compare`：比较两个捕获。
- `vision_diff`：捕获之间的像素级差异。
- `vision_ocr`：从捕获中提取文本。
- `vision_track`：配置区域跟踪。
- `vision_link`：将捕获链接到 AgenticMemory 节点。
- `session_start`：开始视觉会话。
- `session_end`：结束视觉会话。

## 服务器能力检查

```bash
agentic-vision-mcp info
```

agentic-vision-mcp validate
agentic-vision-mcp info
