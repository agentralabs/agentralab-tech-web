## MCP Tools

- `vision_capture`: Capture and embed an image.
- `vision_query`: Query captures by filters.
- `vision_similar`: Find visually similar captures.
- `vision_compare`: Compare two captures.
- `vision_diff`: Pixel-level diff between captures.
- `vision_ocr`: Extract text from a capture.
- `vision_track`: Configure region tracking.
- `vision_link`: Link capture to AgenticMemory node.
- `session_start`: Start a vision session.
- `session_end`: End a vision session.
