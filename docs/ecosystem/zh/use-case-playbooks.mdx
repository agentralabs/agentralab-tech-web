## 4) 三运行时组合栈

当你需要“连续记忆 + 网页感知 + 代码智能”时再组合三者。保持每个运行时可独立安装，可降低故障耦合并提升可调试性。

<Callout type="tip" title="上线策略">
先在生产环境部署单运行时，验证稳定后按顺序逐层叠加。
</Callout>
