## 持久视觉记忆

AgenticVision 将视觉观察结果存储在 `.avis` 中，以便代理可以在会话中重复使用它们。

## 捕获生命周期

1. 捕获图像输入。
2. 生成嵌入。
3. 存储元数据和缩略图。
4. 按时间、描述或相似度查询。

## 神器模型

- `.avis` 是可移植的、本地优先的存储。
- 它专为在代理工作流程中快速读取/查询而设计。

## MCP优先架构

`agentic-vision-mcp` 通过 stdio MCP 公开工具/资源/提示。

任何 MCP 客户端都可以使用相同的接口，而无需特定于供应商的行为。
