AgenticContract was created to solve the fundamental governance gap in AI agent systems.

## The Core Problem

AI agents are increasingly autonomous but lack structured governance frameworks. Without contracts, policies, and limits, agents operate in a trust vacuum where:

1. No rules define acceptable behavior
2. No limits prevent resource exhaustion
3. No approval gates protect high-stakes actions
4. No obligations ensure compliance
5. No audit trail records violations

## How AgenticContract Solves It

AgenticContract provides a complete governance stack:

- **Policies** — Declarative rules (allow/deny/require_approval) at global, session, and agent scope
- **Risk Limits** — Quantitative guardrails with real-time tracking
- **Approvals** — Structured human-in-the-loop workflow
- **Conditions** — Prerequisite checks before action execution
- **Obligations** — Tracked commitments with deadlines
- **Violations** — Severity-classified breach records
