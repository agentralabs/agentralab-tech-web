AgenticCodebase 将源代码编译成可移植的 `.acb` 图表，以进行快速结构查询。

## 你可以做什么

- 编译 Python、Rust、TypeScript 和 Go 存储库。
- 查询符号、依赖关系、影响、调用、相似性、稳定性和耦合性。
- 通过 MCP 使用 `acb-mcp` 公开图形。

＃＃ 人工制品

- 主要工件：`.acb`
- 跨姐妹服务器工作流程可以将 `.acb` 与 `.amem` 和 `.avis` 配对
