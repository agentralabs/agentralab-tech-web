AgenticContract is a policy engine for AI agents, built as a 4-crate Rust workspace.

## Crate Structure

```
agentic-contract/
├── crates/
│   ├── agentic-contract/       # Core library (domain types, engine, SDK traits)
│   ├── agentic-contract-mcp/   # MCP server (JSON-RPC over stdio)
│   ├── agentic-contract-cli/   # CLI binary (acon command)
│   └── agentic-contract-ffi/   # FFI bindings (C/Python interop)
├── python/                     # Python SDK
└── scripts/                    # Install and CI scripts
```

## Data Model

- **Policies**: Rules governing agent behavior (allow/deny/require_approval/audit_only)
- **Risk Limits**: Threshold guardrails with rate/threshold/budget/count types
- **Approvals**: Workflow for controlled actions (rule → request → decision)
- **Conditions**: Conditional execution rules (threshold/time-based/dependency/custom)
- **Obligations**: Mandatory actions with deadlines and status tracking
- **Violations**: Records of policy breaches with severity levels

## SDK Integration

AgenticContract implements all 8 agentic-sdk traits:
- `Sister`, `SessionManagement`, `Grounding`, `Queryable`
- `FileFormatReader`, `FileFormatWriter`, `EventEmitter`, `ReceiptIntegration`
