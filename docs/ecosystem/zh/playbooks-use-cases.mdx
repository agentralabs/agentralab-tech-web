针对常见用户目标的基于步骤的操作手册。

## Playbook 1：本地桌面设置（所有姐妹）

1. 安装工作区依赖项。
2. 运行`./install_all.sh`。
3. 使用 `agentra status` 和 `agentra doctor` 进行验证。
4. 打开 MCP 客户端并确认所有姊妹工具均出现。

## Playbook 2：使用身份验证和工件同步的服务器部署

1. 设置`AGENTRA_RUNTIME_MODE=server` 和`AGENTIC_TOKEN`。
2. 将 `.amem/.acb/.avis` 同步到服务器存储。
3. 运行`agentra server preflight --strict`。
4. 在主机上启动MCP 服务。
5. 验证来自客户端的第一个 MCP 工具调用。

## 手册 3：事件恢复

1. 运行 `agentra backup list` 并选择快照。
2. 运行`agentra backup verify`。
3. 使用范围 (`--memory`、`--mcp`、`--artifacts`) 进行恢复。
4. 重新运行 `agentra doctor` 并验证 MCP 工具。

## 手册 4：受控接管发布

1. Disable one sister with `agentra toggle <sister> off` if needed.
2. 使用 `agentra control off` 释放完全控制。
3. 准备好`agentra control on`后重新启用。
