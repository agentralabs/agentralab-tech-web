使用此页面报告问题、请求功能并共享运行时结果。

## 向何处发送反馈

- 工作空间和编排问题：[代理实验室技术公司](https://github.com/agentralabs/agentralabs-tech/issues)
- AgenticMemory问题：[主体记忆](https://github.com/agentralabs/agentic-memory/issues)
- AgenticCodebase issues: [agentic-codebase](https://github.com/agentralabs/agentic-codebase/issues)
- AgenticVision问题：[代理视觉](https://github.com/agentralabs/agentic-vision/issues)

## 包含哪些内容

- 操作系统和架构（例如：macOS arm64、Ubuntu x86_64）。
- 安装的配置文件（`desktop`、`terminal` 或 `server`）。
- 失败的确切命令。
- 完整的错误输出。
- `.amem`、`.acb`、`.avis` 是本地同步还是服务器同步。

## 提交前

- 运行`agentra doctor`。
- 如果处于服务器模式，请运行 `agentra server preflight --strict`。
- 查看[工作区操作方法](/docs/zh/workspace-how-to#troubleshooting-matrix)中的故障排除矩阵。

## 社区期望

- 保持报告的可重复性。
- 避免共享秘密、令牌或私人数据集。
- 对于每个不同的缺陷优先选择一个问题。
