在 Apple Silicon 上通过发布版本进行测量。

|运营|典型时间 |
|:--|--:|
|图像捕获（文件 -> 嵌入 -> 存储）| ~47 毫秒 |
|相似性搜索（前 5 名）| ~1-2 毫秒 |
| Visual diff | \<1 ms |
| MCP工具往返 | ~7 毫秒 |

这些数字是有方向性的，取决于硬件、模型路径和图像大小。

运行本地验证：

```bash
cargo test --workspace
cargo build --release
agentic-vision-mcp info
```
