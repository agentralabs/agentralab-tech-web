## 此项目中是否有 `cortex` CLI？

否。公共命令界面是 `agentic-vision-mcp`。

## Vision 使用哪个神器？

`.avis`。

## 它适用于所有 MCP 客户端吗？

是的。使用标准 MCP 服务器条目并在安装后重新启动客户端。

## 服务器运行时可以直接读取笔记本电脑文件吗？

首先将工件同步到服务器存储。

## 安装后用户应该做什么？

- 重新启动他们的 MCP 客户端。
- 运行`agentic-vision-mcp info`。
- 可选择在问题跟踪器中分享反馈。
