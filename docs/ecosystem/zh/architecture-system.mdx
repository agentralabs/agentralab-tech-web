本页介绍了工作区编排和三姐妹如何结合在一起。

## 图层

1. **工作区协调器 (`agentra`)**
：检测已安装的姐妹，控制接管状态，运行医生/预检，并管理备份/恢复。
2. **姐妹运行时间**
：`acb-mcp`、`agentic-memory-mcp`、`agentic-vision-mcp` 公开 MCP 工具和工件。
3. **文物**
：`.acb`（代码图）、`.amem`（认知图）、`.avis`（视觉记忆）。
4. **MCP 客户端**
：任何兼容的桌面/终端/服务器 MCP 客户端都可以连接。

## 运行时间模式

- **桌面/终端**：本地工件和本地 MCP 服务器。
- **服务器**：经过身份验证的 MCP 以及同步工件；服务器无法直接读取笔记本电脑本地文件。

## 控制模型

- 当姐妹/文物存在时，自动检测可以进行接管。
- 用户无需卸载即可释放或重新启用接管。
- 每个姐妹的覆盖是非破坏性的并且运行时安全的。

## 规范保证

- 每个姐妹仍然可以独立安装。
- 姐妹之间没有严格的运行时依赖性。
- Web 文档是从规范源文档生成的。
