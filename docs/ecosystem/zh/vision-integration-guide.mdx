## MCP（推荐）

AgenticVision 通过 MCP 使用。任何 MCP 客户端都可以使用相同的服务器条目。

```json
{
  "mcpServers": {
    "agentic-vision": {
      "command": "$HOME/.local/bin/agentic-vision-mcp",
      "args": ["serve"]
    }
  }
}
```

添加条目后，重新启动MCP客户端。

## 使用 AgenticMemory

运行两个服务器以将视觉捕获链接到认知节点：

```json
{
  "mcpServers": {
    "agentic-memory": {
      "command": "$HOME/.local/bin/agentic-memory-mcp",
      "args": ["serve"]
    },
    "agentic-vision": {
      "command": "$HOME/.local/bin/agentic-vision-mcp",
      "args": ["serve"]
    }
  }
}
```

使用 `vision_link` 和内存节点 ID 将代理看到的内容与它记住的内容联系起来。

## 服务器运行时

云服务器无法直接读取笔记本电脑本地工件。首先将 `.avis/.amem/.acb` 同步到服务器存储，然后在那里启动 MCP 服务。
