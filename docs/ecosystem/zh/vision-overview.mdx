AgenticVision 通过便携式 `.avis` 工件和 MCP 服务器为代理提供持久的视觉记忆。

## 它的作用

- 从文件、base64 有效负载、屏幕截图和剪贴板捕获图像。
- 将嵌入和元数据存储在 `.avis` 中。
- 支持视觉搜索、比较、差异、OCR 和内存链接。
- 通过 `agentic-vision-mcp` 将工具公开给任何 MCP 兼容客户端。

＃＃ 人工制品

- 主要工件：`.avis`
- 跨姐妹服务器工作流程可以将 `.avis` 与 `.amem` 和 `.acb` 配对
