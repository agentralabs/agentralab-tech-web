# Installation

## One-command install

```bash
curl -fsSL https://agentralabs.tech/install/comm | bash
```

Installs the `acomm` CLI and `agentic-comm-mcp` MCP server. Configures Claude Desktop and Claude Code automatically.

### Profile-based installs

```bash
# Desktop — configures Claude Desktop, Cursor, Windsurf
curl -fsSL https://agentralabs.tech/install/comm/desktop | bash

# Terminal — configures Claude Code only
curl -fsSL https://agentralabs.tech/install/comm/terminal | bash

# Server — token-based auth, no desktop config
curl -fsSL https://agentralabs.tech/install/comm/server | bash
```

## Source install (Cargo)

```bash
cargo install agentic-comm agentic-comm-mcp
```

Requires Rust 1.70+.

## Python SDK

```bash
pip install agentic-comm
```

Requires Python 3.10+ and the `acomm` binary.

## npm package

```bash
npm install @agenticacomm/comm
```

WebAssembly bindings for Node.js and browser environments.

## Verification

```bash
acomm --version
agentic-comm-mcp --help
```
