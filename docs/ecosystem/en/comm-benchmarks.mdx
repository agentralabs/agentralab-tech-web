# Benchmarks

## Test environment

- **Framework**: Criterion.rs 0.5
- **Statistical method**: Median of 5000+ iterations
- **Confidence**: 95%

## Performance summary

| Operation | Latency | Notes |
|-----------|---------|-------|
| Store creation | 0.02 ms | Create empty `.acomm` file |
| Channel create | 0.03 ms | Add named channel |
| Send message | 0.01 ms | Append to channel |
| Receive message | 0.01 ms | Read from channel |
| Peek message | 0.01 ms | Read without consuming |
| Channel list | 0.05 ms | List all channels |
| Subscribe | 0.02 ms | Add subscription |
| Search 1K msgs | 0.3 ms | Full-text search |
| Search 10K msgs | 2.1 ms | Full-text search |
| Broadcast | 0.05 ms | Send to all channels |
| Store info | 0.01 ms | Read store statistics |
| Health check | 0.02 ms | Validate integrity |

## Performance tiers

- **Sub-millisecond**: All core messaging operations (send, receive, peek, subscribe)
- **Low millisecond**: Search operations scale linearly with message count
- **Negligible overhead**: Channel management and store operations
