## Runtime screenshots

The runtime is split into very small chunks. For each chunk, do this in order:
1. Copy the command.
2. Run it.
3. Compare your output.
4. Check the screenshot.

<Callout type="tip" title="How to read this">
Read each chunk in order. Every chunk has command, output, screenshot, and simple explanation.
</Callout>
