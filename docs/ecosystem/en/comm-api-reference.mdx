# API Reference

## Core modules

### store

The `CommStore` is the top-level API for all communication operations.

```rust
use agentic_comm::CommStore;

let store = CommStore::open("agents.acomm")?;
let store = CommStore::create("new.acomm")?;
```

**Methods:**

| Method | Description |
|--------|-------------|
| `CommStore::create(path)` | Create a new empty store |
| `CommStore::open(path)` | Open an existing store |
| `store.create_channel(name, description)` | Create a channel |
| `store.delete_channel(name)` | Delete a channel |
| `store.list_channels()` | List all channels |
| `store.send(channel, content, sender)` | Send a message |
| `store.receive(channel, limit)` | Receive messages |
| `store.peek(channel, limit)` | Preview without consuming |
| `store.subscribe(channel, agent_id)` | Subscribe agent |
| `store.unsubscribe(channel, agent_id)` | Unsubscribe agent |
| `store.subscriptions(channel)` | List subscriptions |
| `store.search(query, limit)` | Full-text search |
| `store.history(channel, limit)` | Channel history |
| `store.broadcast(content, sender)` | Send to all channels |
| `store.poll(agent_id)` | Poll subscribed channels |
| `store.info()` | Store statistics |
| `store.health()` | Health check |
| `store.save()` | Persist to disk |

### channel

```rust
pub struct Channel {
    pub name: String,
    pub description: Option<String>,
    pub message_count: u64,
    pub subscriber_count: u32,
    pub created_at: u64,
}
```

### message

```rust
pub struct Message {
    pub id: u64,
    pub channel: String,
    pub content: String,
    pub sender: String,
    pub timestamp: u64,
}
```

### subscription

```rust
pub struct Subscription {
    pub channel: String,
    pub agent_id: String,
    pub created_at: u64,
}
```

### error

```rust
pub enum CommError {
    IoError(std::io::Error),
    ChannelNotFound(String),
    ChannelAlreadyExists(String),
    InvalidFormat(String),
    StoreLocked(String),
}
```

All errors implement `std::error::Error` and `Display`.
