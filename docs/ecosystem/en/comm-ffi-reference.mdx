AgenticComm exposes a C-compatible FFI layer through the `agentic-comm-ffi` crate. This enables integration from any language that supports C function calls (Python ctypes, Node.js ffi-napi, Ruby FFI, Go cgo, etc.).

## Shared Library

Build the shared library:

```bash
cargo build --release -p agentic-comm-ffi
# Output: target/release/libagentic_comm_ffi.{so,dylib,dll}
```

## Functions

### `acomm_store_create`

Create a new empty communication store.

```c
AcommHandle* acomm_store_create(const char* path);
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `path` | `const char*` | Path to `.acomm` file (must not exist) |

**Returns:** Opaque handle pointer, or `NULL` if file already exists.

### `acomm_store_open`

Open an existing communication store.

```c
AcommHandle* acomm_store_open(const char* path);
```

**Returns:** Opaque handle pointer, or `NULL` on error.

### `acomm_store_close`

Close and free a store handle.

```c
void acomm_store_close(AcommHandle* handle);
```

### `acomm_store_save`

Persist in-memory changes to disk.

```c
int acomm_store_save(AcommHandle* handle);
```

**Returns:** `0` on success, `-1` on error.

### `acomm_store_info`

Get store statistics as a JSON string.

```c
const char* acomm_store_info(AcommHandle* handle);
```

**Returns:** JSON string (caller must free with `acomm_free_string`). `NULL` on error.

### `acomm_channel_create`

Create a new channel.

```c
int acomm_channel_create(AcommHandle* handle, const char* name);
```

**Returns:** `0` on success, `-1` on error.

### `acomm_channel_list`

List all channels as a JSON array.

```c
const char* acomm_channel_list(AcommHandle* handle);
```

**Returns:** JSON string (caller must free). `NULL` on error.

### `acomm_channel_delete`

Delete a channel and its messages.

```c
int acomm_channel_delete(AcommHandle* handle, const char* name);
```

**Returns:** `0` on success, `-1` on error.

### `acomm_send`

Send a message to a channel.

```c
int64_t acomm_send(
    AcommHandle* handle,
    const char* channel,
    const char* content,
    const char* sender
);
```

**Returns:** Message ID on success, `-1` on error.

### `acomm_receive`

Receive messages from a channel as a JSON array.

```c
const char* acomm_receive(AcommHandle* handle, const char* channel, int limit);
```

**Returns:** JSON string (caller must free). `NULL` on error.

### `acomm_search`

Search messages across channels.

```c
const char* acomm_search(AcommHandle* handle, const char* query, int limit);
```

**Returns:** JSON string (caller must free). `NULL` on error.

### `acomm_broadcast`

Send a message to all channels.

```c
int acomm_broadcast(AcommHandle* handle, const char* content, const char* sender);
```

**Returns:** `0` on success, `-1` on error.

### `acomm_subscribe`

Subscribe an agent to a channel.

```c
int acomm_subscribe(AcommHandle* handle, const char* channel, const char* agent);
```

**Returns:** `0` on success, `-1` on error.

### `acomm_free_string`

Free a string returned by other FFI functions.

```c
void acomm_free_string(const char* s);
```

### `acomm_last_error`

Get the last error message.

```c
const char* acomm_last_error(void);
```

**Returns:** Error string (caller must NOT free; valid until next FFI call). `NULL` if no error.

## Example: Python ctypes

```python
import ctypes
import json

lib = ctypes.CDLL("libagentic_comm_ffi.dylib")

lib.acomm_store_create.restype = ctypes.c_void_p
lib.acomm_store_create.argtypes = [ctypes.c_char_p]

lib.acomm_send.restype = ctypes.c_int64
lib.acomm_send.argtypes = [ctypes.c_void_p, ctypes.c_char_p, ctypes.c_char_p, ctypes.c_char_p]

lib.acomm_receive.restype = ctypes.c_char_p
lib.acomm_receive.argtypes = [ctypes.c_void_p, ctypes.c_char_p, ctypes.c_int]

lib.acomm_store_close.argtypes = [ctypes.c_void_p]

handle = lib.acomm_store_create(b"agents.acomm")
lib.acomm_channel_create(handle, b"tasks")
lib.acomm_send(handle, b"tasks", b"Build the frontend", b"planner")

messages_json = lib.acomm_receive(handle, b"tasks", 10)
messages = json.loads(messages_json)
print(messages)

lib.acomm_store_close(handle)
```

## Example: C

```c
#include "agentic_comm_ffi.h"

AcommHandle* store = acomm_store_create("agents.acomm");
acomm_channel_create(store, "tasks");
acomm_send(store, "tasks", "Build the frontend", "planner");

const char* messages = acomm_receive(store, "tasks", 10);
printf("%s\n", messages);
acomm_free_string(messages);

acomm_store_close(store);
```

## Thread Safety

All FFI functions are thread-safe when called with different handles. Concurrent access to the same handle from multiple threads requires external synchronization.
