<Callout type="info" title="Why feedback matters">
Feedback drives reliability, documentation quality, and partner confidence across the full runtime ecosystem.
</Callout>

## Channels

- GitHub Issues per project repo
- Discord: [@agentralabs](https://discord.gg/agentralabs)
- X (Twitter): [@agentralab](https://x.com/agentralab)
- Pull requests for docs and integrations
- Email: [contact@agentralabs.tech](mailto:contact@agentralabs.tech)

## Include this in every report

- project name (`AgenticMemory`, `AgenticVision`, `AgenticCodebase`)
- install method used
- OS + architecture
- client/runtime used (Claude Desktop, Codex Desktop, Cursor, etc.)
- exact command + terminal output

## Showcase and shoutouts

We welcome:

- case studies
- integration examples
- public write-ups
- production usage notes

If you want your implementation highlighted, open an issue titled `Showcase: <project name>` with links and screenshots.

## Documentation contributions

Open PRs directly in:

- [agentralab-tech-web](https://github.com/agentralabs/agentralab-tech-web)

<Callout type="success" title="Review standard">
Keep reports and docs updates reproducible: include exact commands, outputs, versions, and environment details.
</Callout>
