# Quickstart

## 1. Install

```bash
curl -fsSL https://agentralabs.tech/install/comm | bash
```

This installs the `acomm` CLI and `agentic-comm-mcp` server, configures Claude Desktop and Claude Code.

## 2. Create a store

```bash
acomm create my-agents.acomm
```

## 3. Create a channel and send a message

```bash
acomm channel my-agents.acomm "task-queue"
acomm send my-agents.acomm "task-queue" "Build the login page with React"
```

## 4. Receive messages

```bash
acomm receive my-agents.acomm "task-queue"
```

## 5. Search across all messages

```bash
acomm search my-agents.acomm "login"
```

## MCP integration

Once connected via MCP, ask your LLM:

> "Create a channel called 'tasks' and send a message about building the frontend."

The LLM will call `comm_channel_create` and `comm_send` automatically.

## Python SDK

```bash
pip install agentic-comm
```

```python
from agentic_comm import CommStore

store = CommStore("my-agents.acomm")
print(store.info())
```
