The `acomm` CLI provides command-line access to AgenticComm communication stores.

## Global Options

| Option | Description |
|--------|-------------|
| `--store <path>` | Path to `.acomm` file (default: `~/.comms.acomm`) |
| `--format <fmt>` | Output format: `text`, `json`, `table` (default: `text`) |
| `-h, --help` | Print help information |
| `-V, --version` | Print version |

## Commands

### `acomm channel`

Manage communication channels.

```bash
# Create a channel
acomm channel create "dev-tasks" --description "Development task queue"

# List all channels
acomm channel list

# Delete a channel
acomm channel delete "dev-tasks"

# Show channel details
acomm channel info "dev-tasks"
```

### `acomm send`

Send a message to a channel.

```bash
# Send a simple message
acomm send "dev-tasks" "Build the login page with React"

# Send with explicit sender
acomm send "dev-tasks" "PR #42 ready for review" --sender "developer-agent"

# Send from stdin
echo "Deployment complete" | acomm send "status" --stdin
```

### `acomm receive`

Receive messages from a channel.

```bash
# Receive latest messages (default: 10)
acomm receive "dev-tasks"

# Receive with limit
acomm receive "dev-tasks" --limit 50

# Receive as JSON
acomm receive "dev-tasks" --format json

# Receive with timestamps
acomm receive "dev-tasks" --timestamps
```

### `acomm peek`

Read messages without consuming them.

```bash
# Peek at latest messages
acomm peek "dev-tasks"

# Peek with limit
acomm peek "dev-tasks" --limit 5
```

### `acomm search`

Search messages across channels.

```bash
# Full-text search
acomm search "authentication"

# Search in specific channel
acomm search "authentication" --channel "dev-tasks"

# Search with limit
acomm search "deploy" --limit 20
```

### `acomm broadcast`

Send a message to all channels.

```bash
# Broadcast a message
acomm broadcast "System maintenance at 02:00 UTC"

# Broadcast with sender
acomm broadcast "v1.2.0 deployed to production" --sender "deploy-agent"
```

### `acomm subscribe`

Manage channel subscriptions.

```bash
# Subscribe an agent to a channel
acomm subscribe "dev-tasks" --agent "worker-agent"

# List subscriptions for a channel
acomm subscribe list "dev-tasks"

# Unsubscribe
acomm unsubscribe "dev-tasks" --agent "worker-agent"
```

### `acomm stats`

Print store statistics.

```bash
acomm stats
# Output:
#   Channels:      8
#   Messages:    142
#   Subscriptions: 5
#   File:         ~/.comms.acomm
#   Size:         12.4 KB
```

### `acomm export`

Export communication data.

```bash
# Export all as JSON
acomm export --format json > comms.json

# Export specific channel
acomm export --channel "dev-tasks" --format json
```

### `acomm import`

Import communication data.

```bash
# Import from JSON
acomm import comms.json

# Import with merge (default)
acomm import comms.json --merge

# Import replacing existing
acomm import comms.json --replace
```

### `acomm validate`

Validate a `.acomm` file.

```bash
acomm validate agents.acomm
# Output: Valid .acomm file (v1, 8 channels, 142 messages)
```

### `acomm info`

Print file metadata.

```bash
acomm info agents.acomm
# Output:
#   Format:    ACOM v1
#   Created:   2026-02-20T10:00:00Z
#   Modified:  2026-02-28T14:30:00Z
#   Channels:  8
#   Messages:  142
#   Size:      12.4 KB
```

### `acomm health`

Run integrity checks on the store.

```bash
acomm health
# Output:
#   Header checksum: OK
#   Channel index:   OK
#   Message index:   OK
#   Content blocks:  OK
#   Status: Healthy
```

### `acomm compact`

Reclaim disk space from deleted channels and messages.

```bash
acomm compact
# Output: Compacted 3.2 KB (removed 2 deleted channels, 14 orphaned messages)
```
